# Software-Defined Passive Radar: High-Performance Drone Detection at the Edge

A cutting-edge software-defined passive radar system designed for real-time, high-performance drone detection and classification at the edge. This project leverages passive radar techniques to detect unmanned aerial vehicles (UAVs) without active electromagnetic transmissions, making it ideal for covert surveillance, security operations, and autonomous systems monitoring.

## Key Features

- **Passive Detection**: No radar emissions—detects drones using existing RF signals (TV, cellular, WiFi)
- **Edge Computing**: Optimized for deployment on resource-constrained edge devices and IoT platforms
- **High Performance**: Real-time signal processing with minimal latency
- **Software-Defined**: Flexible, configurable radar processing pipeline adaptable to various RF environments
- **Advanced Signal Processing**: Multi-channel correlation, Doppler analysis, and target tracking

## Applications

- Airspace security and drone detection
- Critical infrastructure protection
- Autonomous vehicle monitoring
- Radio frequency spectrum analysis
- Research in passive radar signal processing

This project combines passive radar theory, signal processing, and edge computing to create a practical, deployable system for next-generation airspace awareness.
